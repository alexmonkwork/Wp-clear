<p>A simple plugin removes unnecessary code from wp_head. Allows you to disable emoji and wp-embed</p>
<p>Required PHP 7.*</p>

<h2><a id="user-content-introduction" class="anchor" aria-hidden="true" href="#introduction"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Description </h2>
<p>Remove Unnecessary Code from wp_head.</p>
<ol>
	<li>WordPress generator meta tag</li>
	<li>Post Relational Links</li>
	<li>Post shortlinks</li>
	<li>Canonical links</li>
	<li>Post and Comment Feed and Other feeds</li>
	<li>Resource hints</li>
</ol>
<p>Remove emoji</p>
<p>Note: Emoticons will still work and emoji’s will still work in browsers which have built in support for them.
This plugin simply removes the extra code bloat used to add support for emoji’s in older browsers.</p>

<p>Disable wp-embed</p>
<ol>
	<li>Prevents others from embedding your site.</li>
	<li>Prevents you from embedding other non-whitelisted sites.</li>
	<li>Disables all JavaScript related to the feature.</li>
	<li>Removes support for the WordPress embed block in the new block editor.</li>
</ol>

	<li>Upload "wp-clear" to the "/wp-content/plugins/" directory</li>
	<li>Activate the plugin through the 'Plugins' menu in WordPress</li>
